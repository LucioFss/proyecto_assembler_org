# proyecto_assembler_org
Este repositorio almacena la resolucion de un proyecto de la materia "Organizacion de Computadoras".

Debe tenerse en consideración que *el proyecto consta de dos partes*: 
la primera es una maquina que simula un procesador microprogramado y la memoria, donde tiene varias instrucciones pre hechas, en esta maquina debiamos de realizar una instruccion que simulara el algoritmo de la multiplicacion como lo haria una ALU real, y plantear 3 programas probando la instruccion.
