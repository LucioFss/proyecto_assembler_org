# proyecto_assembler_org
Este repositorio almacena la resolucion de un proyecto de la materia "Organizacion de Computadoras".

Debe tenerse en consideración que *el proyecto consta de dos partes*: <br />
**La primera** es una maquina que simula un procesador microprogramado y la memoria, donde tiene varias instrucciones pre hechas, en esta maquina debiamos de realizar una instruccion que simulara el algoritmo de la multiplicacion como lo haria una ALU real, y plantear 3 programas probando la instruccion. Ademas debe utilizarse CPUsim para correr esta parte del proyecto.<br />
**La segunda** debemos de realizar un programa en C que utilice una funcion de assembler que sume 2 matrices y las guarde en otra matriz segun el tamaño de las matrices. Para esto utilizamos C y ademas NASM, pero con el entorno de SASM.
